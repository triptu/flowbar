Get Flowbar

Free, open source, runs entirely on your Mac.

Download from GitHub
  1. Open the downloaded Flowbar.dmg and drag Flowbar into your Applications folder.
  2. Launch Flowbar. macOS will show a warning — this is normal for apps that aren't notarized.
  3. Open System Settings → Privacy & Security, scroll down, and click "Open Anyway". You only need to do this once.
  4. That's it! Double-tap Fn to open Flowbar from anywhere.
Why the security warning? Apple charges for a developer certificate to skip this prompt. Flowbar is free and open source — so it's not notarized. The DMG is built automatically by GitHub Actions from the public source code.
